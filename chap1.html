create-react-app

In this chapter we are going to build our first react application

Note: Make sure node.js is downloaded and installed. Also you can use any code editor, in this tutorial I'm going to be using visual studio code.

Step by step instructions

1. Once Node.js is set-up, let's create a folder in the drive where the project files needed to be saved and open that folder in the Vscode.

2. The next step is to open a new terminal in Vscode. Here's where the magic brews.

3. Now to create a new react app, we'll be making use creat-react-app, which is a command-line interface tool which allows us to create and run react applications with no configuration.

npm create-react-app myproject.

4. Now with this, an entire react project is created. Note: myproject is the name of the app.

5. Once the app is ready, it should look something like this:

6. The next step is to run this application on the browser. Let's first navigate inside the project, using the command, 'cd myproject' and then the command 'npm start'.

cd myproject
npm start

7. This command runs the code on the browser under localhost:3000. And the result should look something like this:

8. As the instructions suggest, let's edit App.js under the src folder.

9. By editing the App.js folder, we can bring the desired application. Here I have edited it to 'Hello World'.

Thus, in less than 10 steps, we have created our very first react application. In the next chapter, let's discuss more on the application's folder structure.
